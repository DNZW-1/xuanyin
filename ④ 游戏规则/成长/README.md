## 规划

| 文件 | 说明 |
|------|------|
| 境界体系.md | 筑基→金丹→元婴→化神，各阶要求 |

## 数据流

1. 角色初始境界 → ③ 世界数据库/角色
2. 当前境界/经验 → ⑤ 运行数据/玩家状态
3. 突破判定 → 读取成长规则 → 更新 ⑤ 运行数据

## 编写规则

- 境界体系包含前置条件和小境界划分
- 突破规则包含多种途径（苦修/丹药/双修）
- 正文 >15 行，禁止 stub
