# ④ 游戏规则 — 世界怎么运作

游戏世界中的行为逻辑和机制：战斗怎么算、对话谁先开口、好感怎么涨跌、经济怎么流通。描述"怎么行动"而非"是什么"。

## 目录说明

| 目录 | 内容 |
|------|------|
| 战斗/ | 攻击循环、伤害公式、死亡判定、仇恨机制 |
| 对话/ | 说话风格、信息公开、心理活动规则 |
| 成长/ | 角色升级、境界突破、属性变化 |
| 好感/ | 亲密度、好感度计算、关系阶段变化 |
| 经济/ | 交易系统、货币体系、资源流通 |
| 导演系统/ | AI 叙事节奏控制、伏笔、反转、剧情推进 |
| 事件推进/ | 事件触发条件、分支选择、后果系统 |

## 编写规则

1. 规则文件描述「条件 → 行为 → 后果」的结构化逻辑
2. 每个文件 >15 行正文
3. 规则引用 ③ 世界数据库 中的对象定义
4. 规则读取 ⑤ 运行数据 中的当前状态做决策
