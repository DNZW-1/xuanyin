# 战斗 — 战斗规则

战斗系统的行为逻辑：攻击循环、伤害公式、状态判定、胜负条件。

## 规划

| 文件 | 说明 |
|------|------|
| 基础规则.md | 回合制/实时制、行动顺序 |
| 伤害公式.md | 攻击力×技能倍率−防御，暴击/闪避 |
| 双修战斗.md | 双修模式下的特殊战斗机制 |
| 状态效果.md | Buff/Debuff、控制、持续伤害 |

## 编写规则

1. 每个规则文件描述「条件 → 行为 → 后果」
2. 正文 >15 行，禁止 stub
3. 属性数据从 ③ 世界数据库/角色 和 ⑤ 运行数据/玩家状态 读取

## 与 ③ 世界数据库 的关系

- 角色定义提供属性面板（修为、血量、攻防）
- 技能定义提供技能效果
- 战斗规则决定这些值怎么交互计算
