# 导演系统 — AI叙事控制

控制 AI 叙事的节奏、伏笔、反转、高潮设计。这决定了故事讲得好不好。

## 规划

| 文件 | 说明 |
|------|------|
| 节奏控制.md | 叙事速度、场景切换频率、留白时机 |
| 伏笔规则.md | 如何埋设伏笔、什么时候回收 |
| 反转设计.md | 剧情反转的类型和使用时机 |
| 高潮规划.md | 章节高潮的铺垫和爆发 |
| 分支选择.md | 玩家选项如何影响后续叙事 |

## 工作方式

导演规则 → 影响 AI Prompt → 控制生成内容的叙事质量

## 编写规则

1. 规则需具体可执行，不是空泛的"写得好看"
2. 正文 >15 行，禁止 stub
3. 与 ② AI系统/Prompt 联动，导演规则指导 Prompt 设计
