# 对话 — 对话规则

角色的说话方式、信息公开策略、心理活动透露规则。

## 规划

| 文件 | 说明 |
|------|------|
| 对话风格.md | 各角色的说话方式概述 |
| 冷凝霜的对话风格.md | 冷傲师尊的说话模式 |
| 信息公开规则.md | 什么信息能明说、什么只能暗示 |
| 心理活动规则.md | 内心独白何时出现 |

## 编写规则

1. 规则文件描述角色在什么条件下怎么说话
2. 正文 >15 行，禁止 stub
3. 影响 AI 生成对话时的措辞选择

## 与 ② AI系统 的关系

- 对话规则影响 Prompt 中角色扮演部分的提示词
- 规则越具体，AI 输出的角色一致性越高
