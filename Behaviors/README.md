# Behaviors — 行为规则

本目录存放游戏世界中各类「行为逻辑」的描述。不定义对象"是什么"，只定义"怎么行动"。

## 目录说明

| 目录 | 内容 | 示例 |
|------|------|------|
| Character Behaviors/ | 角色行为：性格驱动、决策风格、日常行为准则 | 师尊的冷傲.md、弟子的服从.md |
| Combat/ | 战斗规则：攻击循环、伤害公式、死亡判定、仇恨机制 | 双修战斗.md、灵力对冲.md |
| Dialogue/ | 对话规则：说话风格、信息控制、心理活动透露规则 | 冷凝霜的对话风格.md |
| Economy/ | 经济规则：交易系统、货币体系、资源流通、灵石价值 | 玄阴山灵石经济.md |
| Politics/ | 政治规则：外交、联盟、战争、势力博弈、情报运作 | 宗门间制衡.md |
| AI Directing/ | 导演规则：叙事节奏控制、伏笔埋设、高潮设计、剧情推进 | 夜袭的节奏控制.md |

## 编写规则

1. 行为文件描述「在什么条件下产生什么行为」，而非单纯叙事
2. 格式建议：条件 → 行为 → 后果，或类似的结构化描述
3. 每个文件至少 >15 行正文
4. 行为规则可以被多个 Definitions 引用（多对多关系）

## 互操作

- Behaviors → Definitions：行为规则引用角色/物品/能力等定义
- Behaviors → Runtime：行为规则读取 Runtime 状态做决策
- AI Directing → Prompts：导演规则直接影响 AI 叙事提示词
