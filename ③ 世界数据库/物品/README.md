# 物品 — 物品模板

所有物品的定义：武器、丹药、装备、材料、任务道具。只定义"是什么物品"，不记录数量或位置。

## 规划

| 文件 | 说明 |
|------|------|
| 筑基丹.md | 基础丹药，突破筑基期 |
| 玄阴剑.md | 玄阴山传承法器 |
| 双修古法.md | 残卷，身份特殊道具 |
| 灵石.md | 通用货币单位 |

## 编写规则

1. 每个物品一个 .md 文件，文件名即物品名
2. 内容包含：品阶、类型、效果描述、稀有度
3. 正文 >15 行，禁止 stub
4. 使用 `[[wikilink]]` 关联可持有角色、产出地点

## 模版

```markdown
# 物品名

品阶/类型/外观描述。

效果和用途...

**关联：**
- [[持有者]]
- [[产出地点]]
```
