# 技能 — 能力模板

所有技能、法术、天赋、武学、功法的定义。只定义"什么能力"，不记录等级或熟练度。

## 规划

| 文件 | 说明 |
|------|------|
| 玄阴心法.md | 玄阴山基础功法 |
| 双修古法.md | 冷凝霜修炼的秘法 |
| 凝霜掌.md | 冷凝霜的招牌法术 |

## 编写规则

1. 每个能力一个 .md 文件，文件名即能力名
2. 内容包含：类型（功法/法术/天赋/武学）、效果、修炼条件
3. 正文 >15 行，禁止 stub
4. 运行时的冷却/熟练度归 ⑤ 运行数据/玩家状态

## 与 ④ 游戏规则/战斗 的关系

- 技能定义描述"有什么效果"
- 战斗规则描述"怎么触发和计算效果"
