# 角色 — 角色模板

所有 NPC、玩家、Boss、怪物的初始属性、背景设定、性格描述。只定义"是谁"，不记录当前状态。

## 编写规则

1. 每个角色一个 .md 文件，文件名即角色名
2. 内容包含：修为、性格、背景、外貌、初始关系
3. 运行时的好感/位置变化归 ⑤ 运行数据/NPC状态
4. 正文 >15 行，禁止 stub
5. 使用 `[[wikilink]]` 关联其他角色、地点、事件

## 模板

```markdown
# 角色名

修为/身份/性格一句话概括。

背景故事...

**关联：**
- [[其他角色]]
- [[相关地点]]
- [[相关事件]]
```
