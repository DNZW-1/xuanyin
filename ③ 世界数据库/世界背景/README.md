# 世界背景

本目录存放游戏世界的历史、神话、文化、地理等「设定说明」。这些不直接参与运行逻辑，但为 AI 和玩家提供世界观深度。

## 内容示例

- 玄阴山宗门史：从开山祖师到第一百三十七代山主
- 双修古法的起源：失传的仙术还是魔道邪法？
- 北域修仙界势力格局：玄阴山、天机阁、魔门三足鼎立
- 金丹期修炼详解：境界、瓶颈、突破方法
- 玄阴山年表：从建宗到现在的重大事件

## 编写规则

1. 每个设定一个文件，文件名即主题
2. 可以使用大纲、年表、人物列传等多种叙事形式
3. 通过 `[[wikilink]]` 与 定义库/事件 中的具体内容关联
4. >15 行正文，禁止 stub
5. 世界背景 中的内容视为「设定真理」—— 定义库 和 运行状态 不应与之矛盾

## 与 定义库 的区别

- 定义库：描述"当下存在的具体对象"
- 世界背景：描述"过去发生的"和"世界运行的底层规则"
