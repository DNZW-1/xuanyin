# ③ 世界数据库 — 游戏世界的全部"定义"

存放游戏世界中所有静态数据——角色模板、物品、技能、地点、势力、种族、世界背景。只定义"是什么"，不记录当前状态。

## 目录说明

| 目录 | 内容 |
|------|------|
| 角色/ | 角色模板：NPC、玩家、Boss 的初始属性和设定 |
| 物品/ | 物品模板：武器、丹药、装备、材料、任务道具 |
| 技能/ | 能力模板：技能、法术、天赋、武学、功法 |
| 地点/ | 地点模板：城市、副本、区域、建筑 |
| 势力/ | 势力模板：宗门、国家、组织 |
| 种族/ | 种族模板：人类、精灵、魔兽、妖族 |
| 世界背景/ | 历史、神话、文化、地理、场景叙事 |

## 编写规则

1. 每个角色/物品/技能一个 .md 文件，文件名即唯一标识
2. 正文 >15 行，禁止 stub
3. 使用 Obsidian wikilink 建立关联图谱，每个文件至少 1-2 个 `[[链接]]`
4. 运行时变化归 ⑤ 运行数据 管
