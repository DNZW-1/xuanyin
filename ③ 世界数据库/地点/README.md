# 地点 — 地点模板

所有可探索的地点：城市、宗门、副本、秘境、建筑。只定义"什么地方"，不记录玩家是否到过。

## 规划

| 文件 | 说明 |
|------|------|
| 玄阴山主峰.md | 宗门核心区域 |
| 青木镇.md | 山脚下的小镇 |
| 玄阴山寝殿.md | 冷凝霜的居所 |
| 北域雪原.md | 玄阴山所在区域 |

## 编写规则

1. 每个地点一个 .md 文件
2. 内容包含：位置描述、可交互对象、关联角色、关联事件
3. 正文 >15 行，禁止 stub
4. 玩家是否解锁归 ⑤ 运行数据/玩家状态

## 关联

- 地图渲染从本目录读取数据
- 场景切换通过地点名索引
