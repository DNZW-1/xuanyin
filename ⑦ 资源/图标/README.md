# 图标 — 游戏图标资产

游戏中使用的图标：系统图标、状态图标、物品图标、UI 装饰图标。

## 分类

| 类别 | 说明 | 格式 |
|------|------|------|
| 系统/ | 存档、设置、退出、返回 | SVG/PNG |
| 状态/ | 血量、灵力、Buff、Debuff | SVG/PNG |
| 物品/ | 各物品分类图标 | PNG |
| UI装饰/ | 分隔线、边框、装饰元素 | SVG |

## 命名规范

`类别_名称.svg`，例：`系统_存档.svg`、`状态_血量.svg`

## 现状

当前所有图标用 Emoji 和纯 CSS 代替（⚙️❤️⚔️📦）。
