# ⑦ 资源 — 多媒体资产

游戏使用的所有非代码资源：图片、音乐、音效、模板、图标。

## 目录说明

| 目录 | 内容 |
|------|------|
| 图片/ | 角色立绘、背景图、CG、界面元素 |
| 音乐/ | BGM、环境氛围音乐 |
| 音效/ | SE、点击音、事件音效、环境音 |
| HTML模板/ | 可复用的页面片段、弹窗、UI 布局模板 |
| 图标/ | 系统图标、状态图标、物品图标 |

## 格式约定

- 图片：PNG（立绘/UI）、JPG（背景）
- 音乐：MP3 或 OGG
- 音效：WAV 或 MP3 短片段
- 文件名：`类型-名称.扩展名`

## 现状

暂无资源文件。所有 UI 目前是纯 CSS 绘制（Emoji 代替立绘）。
