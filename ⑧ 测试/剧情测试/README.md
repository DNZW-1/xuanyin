# 剧情测试 — 故事连贯性测试

测试游戏场景的逻辑自洽性、角色行为一致性、分支覆盖完整性。

## 测试内容

| 测试项 | 方法 | 预期 |
|--------|------|------|
| 剧情连贯 | 完整播放一个章节 | 无逻辑跳跃、角色行为合理 |
| 角色一致 | 检查每个角色的台词和行为 | 符合角色设定 |
| 分支覆盖 | 遍历每个选项分支 | 每个分支有独立合理的后续 |
| 链接完整 | 检查所有 [[wikilink]] | 在 Obsidian 中可跳转 |
| 前后矛盾 | 检查 flag 和好感变化 | 事件结果与状态更新一致 |

## 文件格式

每个章节一个测试文件，遍历场景步骤并记录结果。
