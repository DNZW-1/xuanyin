# UI测试 — 界面交互测试

测试游戏界面的交互功能是否正常：存档/读档、对话推进、选项选择、地图导航。

## 测试内容

| 测试项 | 操作 | 预期 |
|--------|------|------|
| 对话推进 | 点击继续按钮/按空格 | 下一句正确显示 |
| 选项分支 | 点击选项 | 对应分支内容出现 |
| 快速存档 | 按 F5 | 存档提示出现 |
| 快速读档 | 按 F8 | 读取最后一次存档 |
| 存档菜单 | 打开存档菜单，操作 | 完整的存档/读档/覆盖/删除 |
| 立绘显示 | 切换活跃角色 | 立绘和名称正确更新 |
| 进度条 | 推进场景 | 头像下方进度条变化 |
| 地图 | 查看地图 | 位置标记正确 |

## 文件格式

每个测试用例一个 .md 或 .json 文件，记录操作步骤和结果。
