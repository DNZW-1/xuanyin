# Prompt测试 — AI输出质量测试

测试 AI 在不同 Prompt 配置下的输出质量、角色一致性、叙事效果。

## 测试内容

| 测试项 | 方法 | 预期 |
|--------|------|------|
| 角色一致性 | 同一 Prompt 多次生成同一场景 | 角色性格、措辞一致 |
| 风格符合度 | 按风格指南生成文字 | NSFW 风格达标 |
| 叙事质量 | 评估节奏、伏笔、高潮 | 故事流畅、有张力 |
| 规则遵守 | 检查是否遵守守则 | 不瞎猜、不臆想 |

## 文件格式

每个测试用例一个 .md 文件：

```markdown
# Prompt测试：[测试名]

## 输入
[Prompt 文本]

## 预期输出
[期望的 AI 输出风格]

## 实际输出
[记录实际输出]

## 结论
[通过/失败 + 改进建议]
```
