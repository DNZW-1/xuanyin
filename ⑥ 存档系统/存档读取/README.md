# 存档读取 — 存档读写逻辑

存档数据的序列化/反序列化逻辑、格式转换、版本兼容处理。

## 功能

| 功能 | 说明 |
|------|------|
| 保存 | 收集运行数据 → 序列化 JSON → 写入文件 |
| 读取 | 读取文件 → 反序列化 → 校验 → 注入运行数据 |
| 版本兼容 | 检查版本号 → 按版本做数据迁移 |
| 校验 | 字段完整性检查、数据一致性校验 |
| 加密 | 可选，存档内容加密保存 |

## 版本管理

save.json 包含 `_protocol.version` 字段。版本不匹配时按版本迁移逻辑转换。

## 数据流

保存：⑤ 运行数据 → ⑥ 存档系统/存档读取 → save.json
读取：save.json → ⑥ 存档系统/存档读取 → ⑤ 运行数据
