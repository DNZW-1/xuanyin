# 快速存档 — 快捷键存档

通过快捷键快速保存/读取，不打开存档菜单。

## 快捷键

| 按键 | 功能 |
|------|------|
| F5 | 快速存档（覆盖上一次快速存档） |
| F8 | 快速读档（读取上一次快速存档） |

## 特点

- 只有一个快速存档槽位
- 每次 F5 覆盖上一次快速存档
- 不中断当前游戏流程
- 存档时显示短暂提示（2 秒后消失）

## 存储

快速存档存为独立文件：`quick_save.json`，位于 ⑥ 存档系统/快速存档/。

## 注意事项

- 快速读档前自动创建一次自动存档，防止误操作
- 战斗中等不可读档状态禁用 F8
