# 剧情节点 — 预设剧情存档点

开发预设的剧情存档点。用于跳转回特定的剧情位置，方便测试和重玩。

## 说明

| 字段 | 说明 |
|------|------|
| node_id | 节点唯一标识 |
| chapter | 所属章节 |
| description | 剧情描述 |
| timestamp | 创建时间 |
| state | 该节点的完整游戏状态 |

## 用途

- 开发者快速跳转到特定剧情位置测试
- 玩家解锁成就后跳转到特定节点
- 章节选择功能的基础数据

## 存储

每个剧情节点一个 JSON 文件，放在本目录下。文件名格式：`node_章节ID_场景名.json`。
