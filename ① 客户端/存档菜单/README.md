# 存档菜单 — 存档/读档界面

游戏中存档/读档的 UI，对接 ⑥ 存档系统。

## 功能

- 存档槽位列表（每个槽显示：章节名、时间、缩略图、级别）
- 手动存档覆盖确认
- 自动存档特殊标记
- 快速存档快捷键提示
- 存档信息预览（日期、游戏内时间、位置）

## 规划文件

| 文件 | 说明 |
|------|------|
| SaveMenu.js | 存档菜单主面板 |
| SaveSlot.js | 单个存档槽组件 |
| ConfirmDialog.js | 覆盖确认弹窗 |

## 快捷键

| 按键 | 功能 |
|------|------|
| F5 | 快速存档 |
| F8 | 快速读档 |
| Escape | 关闭菜单 |
