# 对话框 — 文字和对话系统

角色对话、旁白叙事、选项分支的显示与控制。

## 功能

- 逐字显示文字（打字机效果）
- 角色名显示（颜色区分性别）
- 选项分支（点击选择）
- 历史回顾（查看已读对话）
- 快进跳过（已读内容自动跳过）

## 规划文件

| 文件 | 说明 |
|------|------|
| DialogueBox.js | 对话框主组件 |
| ChoicePanel.js | 选项分支面板 |
| HistoryView.js | 对话历史查看器 |
| SkipToggle.js | 快进模式控制 |

## 数据流

场景数据 → 对话框 → 逐字渲染 → 等待交互 → 下一句
