# 地图 — 场景导航

游戏世界地图的渲染和交互。

## 功能

- 区域展示（山脉、城镇、宗门、副本入口）
- 当前位置标记（闪烁/高亮）
- 已解锁/未解锁区域区分
- 点击区域跳转（触发场景切换）
- 传送阵动画效果

## 规划文件

| 文件 | 说明 |
|------|------|
| MapRenderer.js | 地图渲染（SVG/Canvas） |
| LocationMarker.js | 地点标记组件 |
| MapData.js | 地图节点数据（从 ③ 世界数据库/地点/ 加载） |

## 现状

当前地图为静态 SVG，内嵌在 index.html 中。
