# 背包界面 — 物品管理和查看

玩家背包的 UI 展示和交互。

## 功能

- 物品格子展示（图标 + 数量）
- 物品分类筛选（丹药/武器/材料/任务道具）
- 物品详情面板（描述、属性、使用按钮）
- 装备/卸下操作
- 物品使用（丹药服用、武器切换）

## 规划文件

| 文件 | 说明 |
|------|------|
| Inventory.js | 背包主界面 |
| ItemSlot.js | 单个物品格 |
| ItemDetail.js | 物品详情面板 |
| CategoryFilter.js | 分类筛选标签 |

## 数据源

物品数据从 ③ 世界数据库/物品/ 读取模板，实例数据从 ⑤ 运行数据/当前背包/ 加载。
