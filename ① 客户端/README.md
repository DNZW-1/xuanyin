# ① 客户端 — 玩家看到的东西

前端游戏引擎，玩家直接交互的部分。目前基于纯 HTML/CSS/JS 实现。

## 目录说明

| 目录 | 内容 |
|------|------|
| HTML/ | 入口页面，渲染结构 |
| CSS/ | 样式文件，主题、动画 |
| JavaScript/ | 核心引擎逻辑，场景控制 |
| UI组件/ | 可复用的界面组件 |
| 对话框/ | 文字对话显示、选项分支 |
| 背包界面/ | 玩家背包/物品查看 |
| 地图/ | 场景地图、导航 |
| 存档菜单/ | 存档/读档界面 |

## 现状

`HTML/index.html` 是当前唯一文件，包含内联 CSS 和 JS。后续将逐步拆分为独立模块。
