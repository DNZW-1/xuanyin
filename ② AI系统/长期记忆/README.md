# 长期记忆 — 跨会话保留的信息

跨多个对话会话保留的重要信息。不依赖 AI 上下文窗口。

## 存储方式

| 方式 | 内容 | 更新频率 |
|------|------|---------|
| .opencode/lessons.md | 项目教训和复现步骤 | 每次踩坑后追加 |
| .opencode/instructions | 项目协议和结构 | 结构变更时更新 |
| 各目录 README | 目录说明和约定 | 内容变更时同步 |

## 记录规则

- 每个教训记录：场景 → 错误做法 → 根因 → 正确做法
- 教训必须可复现（含完整命令）
- 不记录敏感信息（token/密码）
- lessons 文件直接写入 .opencode/learned 全局 skill 可选
