# 提示词

本目录存放所有 AI 提示词、系统指令模板、风格指南、以及本项目 AI 协作的约定文件。

## 目录内容说明

| 文件/子目录 | 说明 |
|------------|------|
| 守则.md | 操守五荣五耻及执行细则 |
| 风格指南.md | NSFW 写作风格指南（原根目录迁移至此） |
| 系统提示.md（计划中） | 游戏模式下给 AI 的系统提示词模板 |
| 角色扮演模板.md（计划中） | 各角色的对话风格提示词 |

## 编写规则

1. 提示词按用途分文件，不要把所有提示词塞到一个文件里
2. 每个提示词文件包含明确的「触发条件」和「使用方式」说明
3. 敏感/NSFW 内容的风格指引放在风格指南.md
4. 操作守则放在守则.md，不分拆
5. 所有文件 >15 行正文

## 互操作

- Prompts → Behaviors/AI Directing：导演规则影响提示词设计
- Prompts → Definitions：提示词引用角色定义以生成对话
