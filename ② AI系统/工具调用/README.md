# 工具调用 — 工具使用规则

AI 调用各种工具（git、文件操作、搜索等）的规则和约定。

## 可用工具

| 工具 | 用途 | 使用时机 |
|------|------|---------|
| bash | 执行命令行 | git 操作、文件操作验证 |
| read | 读文件 | 了解内容后再修改 |
| write | 写文件 | 创建新文件 |
| edit | 编辑文件 | 修改已有内容 |
| grep | 搜索内容 | 查找引用和模式 |
| glob | 搜索文件名 | 定位文件位置 |

## 使用规则

- 改代码前先 Read，确认上下文
- 批量操作并行的用多个工具调用
- 敏感信息（token/密码）用后立即清理
- 每一步操作后验证结果
