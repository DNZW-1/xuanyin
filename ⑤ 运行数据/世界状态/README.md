# 世界状态 — 宏观状态数据

游戏世界的宏观运行状态：天气、事件进度、势力关系、世界时钟。

## 已有数据

`save.json` 是存档的唯一真相源，包含：
- `_protocol`：协议版本和项目规则
- `player`：玩家状态
- `relations`：NPC 关系和好感
- `diary`：日记条目（叙事总结）
- `flags`：已解锁/已触发的内容标记
- `world`：世界状态（时间、天气、位置、已知地点）

## 数据更新规则

1. 启动时先读 save.json 重建上下文
2. 对话结束后写回 save.json
3. 存档即唯一真相源
4. 不依赖 AI 上下文记忆

## 规划扩展

| 字段 | 说明 |
|------|------|
| faction_relations | 势力间外交关系值 |
| event_queue | 即将发生的事件列表 |
| global_timer | 世界时间线进度 |
