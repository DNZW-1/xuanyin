# 当前事件 — 正在触发的事件

当前正在触发或进行的特定事件详情。

## 数据字段

| 字段 | 说明 |
|------|------|
| event_name | 事件名 |
| trigger_type | 触发方式（自动/选择/条件） |
| participants | 参与角色列表 |
| choices_made | 已做出的选择 |
| state | 事件状态（进行中/已完成/分支中） |
| outcome_flags | 事件产生的 flag |

## 与 ③ 世界数据库/世界背景 的关系

- 事件叙事文本在 世界背景 中
- 当前事件/ 只记录状态数据
- 场景文件定义了"发生了什么"，本目录记录"进行到哪了"

## 存储格式

推荐 JSON，内容与 save.json 中的 flags 和 diary 联动。
