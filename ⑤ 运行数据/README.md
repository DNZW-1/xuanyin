# ⑤ 运行数据 — 这个存档里的世界

游戏运行时的实时状态数据。每个存档对应一份运行数据，随游戏推进不断变化。

## 目录说明

| 目录 | 内容 |
|------|------|
| 玩家状态/ | 苏尘的属性、境界、背包、位置、Buff、任务 |
| NPC状态/ | 各 NPC 当前好感、位置、生死、特殊状态 |
| 世界状态/ | 天气、局势、事件进度、宗门关系 |
| 当前章节/ | 正在进行的章节、时间线、阶段性状态 |
| 当前事件/ | 当前正在触发的事件详情 |
| 当前背包/ | 当前持有的物品实例 |

## 数据流

1. 启动 → 读 `世界状态/save.json` 重建所有运行数据
2. 运行时 → 各目录实时更新
3. 存盘 → 汇总到 `世界状态/save.json`

## 编写规则

1. 推荐 JSON 格式，机器可读写
2. 每次状态变更必须写回文件，不依赖 AI 上下文记忆
3. 运行数据不包含长叙事文本——叙事在 ③ 世界数据库/世界背景 中
