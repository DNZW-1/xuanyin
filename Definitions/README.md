# Definitions — 定义库

本目录存放游戏世界中所有对象的「模板」定义。只描述对象"是什么"，不记录任何运行时状态。

## 目录说明

| 目录 | 内容 | 示例 |
|------|------|------|
| Characters/ | 角色模板：NPC、玩家、怪物、Boss 的初始属性和设定 | 冷凝霜.md、苏尘.md |
| Items/ | 物品模板：武器、丹药、装备、材料、任务道具 | 筑基丹.md、玄阴剑.md |
| Abilities/ | 能力模板：技能、法术、天赋、武学、功法 | 玄阴心法.md、双修古法.md |
| Factions/ | 势力模板：宗门、国家、组织、公会 | 玄阴山.md、天机阁.md |
| Locations/ | 地点模板：城市、副本、区域、建筑 | 玄阴山主峰.md、青木镇.md |
| Species/ | 种族模板：人类、精灵、魔兽、妖族 | 人族.md、狐妖.md |
| Events/ | 事件模板：随机事件、剧情事件、灾难、奇遇 | 玄阴山夜袭.md |

## 编写规则

1. 每个角色/物品/能力单独一个 .md 文件，文件名即唯一标识
2. 初始状态在定义中给出（如修为、品阶），运行时的动态变化归 Runtime/ 管
3. 内容使用 Obsidian wikilink 建立关联图谱，每个文件至少 1-2 个 `[[链接]]`
4. 禁止空文件 / stub：每篇正文 >15 行

## 互操作

- Definitions → Behaviors：定义引用的行为规则（如角色性格定义引用的 Character Behaviors 文件）
- Definitions → Runtime：定义作为模板，Runtime 复制实例化后记录变化
- Definitions → Lore：跨越多个定义的世界观背景存放在 Lore/
